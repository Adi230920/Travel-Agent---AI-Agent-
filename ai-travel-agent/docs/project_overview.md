# Project Overview — AI Travel Agent

## Purpose
An intelligent multi-agent system that recommends personalized travel 
destinations and generates optimized day-wise itineraries through 
multi-step AI reasoning.

## User
Working professionals and students with limited time for trip planning.

## Core Flow
INPUT → RECOMMENDATION (top 5) → USER SELECTION → ITINERARY GENERATION

## V1 Boundaries
- No auth, no payments, no booking, no maps
- Desktop-first demo UI
- Session memory only (no persistence)
- Free-tier APIs only

## Success Criteria
The agent must demonstrate a full multi-step planning loop:
  recommendation → selection → itinerary
This is a dealbreaker feature. Everything else is secondary.
